
## Descripción
Aplicación simple en Java que demuestra el uso básico de HashMap para almacenar pares clave-valor, en este caso nombres de personas asociados a sus edades.

## Funcionalidades
- Crear un mapa (HashMap) con clave String y valor Integer
- Añadir entradas (nombre → edad)
- Recuperar un valor específico usando su clave
- Mostrar el resultado por pantalla

## Tecnologías
- Java
- HashMap
- Map (interfaz)
